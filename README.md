# Mission School 🚀

یک پلتفرم آموزشی ماجراجویانه برای دانش‌آموزان که درس‌های فارسی، عربی، انگلیسی و علوم را به صورت ماموریت‌های جذاب درمی‌آورد.

## ویژگی‌ها:
- دارای ۴ موضوع درسی (فارسی، عربی، انگلیسی، علوم)
- استفاده از AI (Claude) برای تحلیل متن کتاب‌های درسی و ساخت ماموریت‌های جدید
- سیستم امتیازدهی و مدال‌دهی برای تشویق دانش‌آموزان
- طراحی ریسپانسیو و مدرن

## چطور اجرا کنیم؟
1. ابتدا وابستگی‌ها را نصب کنید:
   ```bash
   npm install
   ```
2. سپس پروژه را روی لوکال اجرا کنید:
   ```bash
   npm run dev
   ```

## ساختار پروژه:
- `src/App.jsx`: کد اصلی برنامه (بازی)
- `src/main.jsx`: نقطه ورود React
- `public/`: دارایی‌های استاتیک
