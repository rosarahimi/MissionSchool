# Mission School 🚀

یک پلتفرم آموزشی ماجراجویانه برای دانش‌آموزان که درس‌های فارسی، عربی، انگلیسی و علوم را به صورت ماموریت‌های جذاب درمی‌آورد.

## ساختار پروژه:
- **Frontend**: محیط کاربری (React + Vite)
- **Backend**: مدیریت دیتابیس و احراز هویت (Node.js + Express + MongoDB)

## ویژگی‌ها:
- دارای ۴ موضوع درسی پیش‌فرض.
- سیستم هوشمند تحلیل متن کتاب با استفاده از هوش‌مصنوعی.
- متصل به دیتابیس MongoDB Atlas.
- تالار افتخارات و سیستم دریافت نشان (Badges).

## چطور اجرا کنیم؟
۱. ابتدا در هر دو پوشه وابستگی‌ها را نصب کنید:
   ```bash
   npm run install-all
   ```
۲. سپس برای اجرای همزمان فرانت‌تند و بک‌تند:
   ```bash
   npm run dev
   ```

## دیتابیس:
اطلاعات دیتابیس در فایل `Backend/.env` ذخیره شده است.

git add . && git commit -m "Fix" && git push origin main
